# accountbook
# 概要
家計簿アプリです。始めた日から毎日何円使ったかを、一日ごと、一週間ごと、一カ月ごとの合計を計算します。
# 特徴
合計からそれぞれ何にお金を使ったかのパーセンテージで計算し、グラフを作成します。
# 使い方
* npx sequelize*cli db:migrade --env development -ローカル環境でデータベースファイルを作成するコマンド。
* npm start - プログラムを開始するコマンドです。
